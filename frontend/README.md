# Nichefy Frontend

Next.js 14 frontend for the Nichefy application.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

The app will be available at `http://127.0.0.1:3000`

## Features

- Search for niche artists based on a popular artist
- Beautiful, modern UI with Tailwind CSS
- Responsive design for mobile and desktop
- Real-time artist recommendations

## Getting Spotify Credentials

To use this app, you'll need:
1. **Spotify Artist ID**: Found in the Spotify artist URL (e.g., `spotify.com/artist/[ID]`)
2. **Spotify Access Token**: Get from Spotify Web API using OAuth or Client Credentials flow

You can get an access token using:
- Spotify Web API Console: https://developer.spotify.com/console/
- Or implement OAuth flow in your app
